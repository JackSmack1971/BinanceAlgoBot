# API Contracts

## 1. Strategy Interface (`strategy.py`)

| Method             | Input Parameters                                                                                                | Output                                  |
| ------------------ | ------------------------------------------------------------------------------------------------------------- | --------------------------------------- |
| `__init__`         | `client` (`object`): Binance API client.                                                                     | None                                    |
|                    | `symbol` (`str`): Trading symbol.                                                                           |                                         |
|                    | `interval` (`str`): Timeframe interval.                                                                     |                                         |
|                    | `initial_balance` (`float`): Initial trading balance.                                                        |                                         |
|                    | `risk_per_trade` (`float`): Risk per trade as a fraction of the balance.                                     |                                         |
| `calculate_indicators` | None                                                                                                        | `pd.DataFrame`: DataFrame with calculated indicators. |
| `run`              | None                                                                                                        | `pd.DataFrame`: DataFrame with trading signals.        |
| `open_position`    | `side` (`str`): "buy" or "sell".                                                                            | None                                    |
|                    | `price` (`float`): Entry price.                                                                              |                                         |
|                    | `size` (`float`): Position size.                                                                             |                                         |
| `close_position`   | `price` (`float`): Exit price.                                                                               | None                                    |

## 2. ExchangeInterface (`exchange_interface.py`)

| Method             | Input Parameters                                                                                                | Output                                  |
| ------------------ | ------------------------------------------------------------------------------------------------------------- | --------------------------------------- |
| `fetch_market_data` | `symbol` (`str`): The trading symbol.                                                                        | None (Raises `NotImplementedError`)     |
| `place_order`      | `symbol` (`str`): The trading symbol.                                                                        | None (Raises `NotImplementedError`)     |
|                    | `side` (`str`): "buy" or "sell".                                                                            |                                         |
|                    | `quantity` (`float`): The quantity to trade.                                                                 |                                         |
|                    | `order_type` (`str`): The type of order (e.g., "market", "limit").                                           |                                         |
|                    | `price` (`float`, optional): The price at which to place the order (for limit orders).                         |                                         |
| `get_account_balance` | None                                                                                                        | None (Raises `NotImplementedError`)     |
| `get_order_status` | `order_id` (`str`): The ID of the order.                                                                      | None (Raises `NotImplementedError`)     |
